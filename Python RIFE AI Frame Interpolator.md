# Python-RIFE-AI-Frame-Interpolator
 - Adds Frames to Videos and Gifs

Run the "Install Requirements" file

Put your video in the MEDIA folder

Run the "Start Adding Frames", or the "Start Removing Frames" file
add will let you multiply framerate by a value of your choice
remove will let you set the fps of your video if you want a very specific framerate

Choose a number when the console prompts you to

When finished, your export is in the EXPORT folder
